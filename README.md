# AED地図案内アプリ

現在地から最適なAED設置施設を検索・案内するWebシステム

## 🚀 概要
「いざというときに迷わずAEDを取りに行くための道具」をコンセプトとした、位置情報特化型のWebアプリです。生コードからの脱却とモダンなWebシステム開発の学習を目的としているため、規模に見合わない部分がありながらも、フレームワーク等の技術を採用しています。

## 🛠 技術スタック

### Backend
- **Laravel 12**: ビジネスロジック及び認証基盤
- **PostgreSQL / PostGIS**: 地理空間データの管理

### Frontend
- **React / TypeScript**: コンポーネント指向による堅牢なUI開発
- **Inertia.js**: 
- **Vite**: 高速なフロントエンドビルド環境

### Infrastructure / API
- **Docker (Laravel Sail)**: 開発環境のコンテナ化
- **Google Maps Platform**: 地図描画・ルート案内
- **@vis.gl/react-google-maps**: React向けGoogle Mapsライブラリ

## ✨ こだわり・学習のポイント

- **地理空間情報の高度な扱い**
  単なる座標保存ではなく、PostGISを採用することで「半径〇m以内のAEDを抽出」といった計算をデータベース側で効率化しました。
- **型安全な開発**
  TypeScriptを導入し、バックエンドから渡るデータの型定義を行うことで、バグの抑制と開発速度の向上を図っています。
