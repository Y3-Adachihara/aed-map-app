# AED地図案内アプリ

現在地から最適なAED設置施設を検索・案内するWebシステム

## 📝概要

「いざというときに迷わずAEDを取りに行くための道具」をコンセプトとした、位置情報特化型のWebアプリです。生コードからの脱却とモダンなWebシステム開発の学習を目的としているため、規模に見合わない部分がありながらも、フレームワーク等の技術を採用しています。

## ⚙機能一覧（計画）

### システムロール

- 一般ユーザ
- 管理者

### 最低要件

- 一般ユーザ
    - ログイン・ログアウト
    - ユーザ登録・退会
    - 設置場所の一覧表示
    - 選択した設置場所までの案内
- 管理者
    
    ユーザログ参照
    

### 願望要件

- 一般ユーザ
    
    履歴参照
    
- 管理者
    
    設置場所データの登録・編集・削除
    

## 🛠 技術スタック

- Laravel 12
- PostGIS（PosgreSQL）
- React
- Inertia.js
- Docker(Laravel Sail)
- Google Maps API

### 備考

　GoogleMapsのライブラリとして、**@vis.gl/react-google-maps** を使用しています。

## ✨ こだわり・学習のポイント

- **地理空間情報の高度な扱い**
単なる座標保存ではなく、「半径〇m以内のAEDを抽出」といった計算をデータベース側で効率化するため、Laravel標準のMySQLではなく、PostGISを採用しています。
- **型安全な開発**
TypeScriptを導入し、バックエンドから渡るデータの型定義を行うことで、バグの抑制と開発速度の向上を図っています。
